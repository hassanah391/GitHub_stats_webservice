About This Project

The GitHub Stats Web Service was inspired by my passion for open-source development and the need for quick, insightful statistics on GitHub repositories. I wanted to create a simple tool that developers could use to track their project growth and engagement with ease. This project started as a personal initiative and evolved into a Portfolio Project for ALX. You can find more details about my journey and contributions in the links below.
